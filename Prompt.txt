You are a senior software architect and Python engineer.

Your task is to build a clean production-quality implementation of AQF (Adaptive Query Forms) based on the supplied research paper and existing AQF prototype implementations.

The objective is NOT to reproduce the existing codebase.

The objective is to create a clean reviewer-ready AQF Demo System suitable for:

* Thesis defense
* Journal reviewers
* End-user usability studies
* Demonstration videos

---

## PROJECT GOAL

Build a complete AQF Demo System capable of:

1. Loading ORBDA/openEHR JSON datasets
2. Building Schema Graphs
3. Computing Queriability Scores
4. Performing Candidate Selection
5. Building Canonical Structures
6. Applying Operator-Aware Field Classification
7. Generating Adaptive Query Forms
8. Translating forms into AQL
9. Showing the entire AQF pipeline visually
10. Collecting usability-study metrics

The architecture must follow the AQF paper exactly:

Dataset
↓
Schema Graph
↓
Queriability
↓
Candidate Selection
↓
Canonical Structure
↓
Operator Awareness
↓
Adaptive Forms
↓
Query Realization (AQL)

## SERVICES

1. schema_graph.py

Create:

SchemaGraphBuilder

Responsibilities:

* Load ORBDA/openEHR JSON
* Extract hierarchical paths
* Build containment edges
* Build co-occurrence edges
* Infer datatypes
* Generate schema graph

Export:

schema_graph.json

---

2. queriability.py

Create:

QueriabilityEngine

Implement paper equations:

LU(v)

SC(u,v)

Q(v)

Where:

LU(v) = coverage × diversity

Q(v) = LU(v) + μ Σ(SC × LU)

Export:

weighted_graph.json

node_scores.json

---

3. candidate_selection.py

Create:

CandidateSelector

Implement:

score(v) >= θ × maxscore

Requirements:

* Preserve parent chain
* Prevent orphan nodes
* Maintain containment integrity

Export:

reduced_graph.json

---

4. canonical_structure.py

Create:

CanonicalStructureBuilder

Convert reduced graph into:

Form Groups
Form Subgroups
Form Elements

Output:

CanonicalTree

CanonicalForm

Export:

canonical_tree.json

canonical_form.json

---

5. operator_awareness.py

Create:

OperatorAwareness

Create datatype-to-operator mapping:

NUMERIC

= != > >= < <=

TEXT

= != contains starts_with

BOOLEAN

=

DATE

before
after
between
=======

Annotate all form elements.

Export:

operator_map.json

---

6. adaptive_forms.py

Create:

AdaptiveFormGenerator

Implement:

Utility:

U(F)

Complexity:

C(F)

Constraint:

C(F) <= κ

Default:

κ = 53

Output:

AdaptiveForm

Export:

adaptive_form.json

---

7. query_realizer.py

Create:

QueryRealizer

Responsibilities:

Convert:

field
operator
value

into:

Predicate
QueryPlan
AQL

Example:

Age > 50

becomes:

SELECT e
FROM EHR e
WHERE e/demographic/age > $p0

Output:

query_explanation.json
